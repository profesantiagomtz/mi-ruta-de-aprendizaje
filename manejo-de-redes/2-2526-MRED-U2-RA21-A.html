A-01

¿Qué es el Sistema Operativo Internetwork IOS?

El IOS es el sistema operativo que usan muchos equipos Cisco para funcionar. Así como una computadora necesita Windows o Linux para trabajar, un router o switch necesita IOS para recibir comandos, administrar interfaces, guardar configuraciones y mover datos por la red.

Explicación clara

Piensa en el router como si fuera una computadora especializada. El hardware es la “máquina”, pero el IOS es el programa que permite decirle qué hacer: ponerle nombre, asignar IP, activar puertos, proteger accesos y revisar si todo funciona.

Función del IOS¿Para qué sirve?
Control del hardwareAdministra interfaces, memoria, puertos y procesos internos.
ConfiguraciónPermite aplicar comandos para modificar el comportamiento del equipo.
SeguridadPermite configurar contraseñas, banners, acceso remoto y privilegios.
ConectividadPermite que el dispositivo participe en la comunicación de red.
A-02

Métodos de acceso al IOS

Para configurar un router o switch primero debes entrar a su interfaz de comandos. No todos los accesos son iguales: algunos son físicos, otros remotos y otros casi no se usan actualmente.

MétodoTipoIdea principal
ConsolaFísicoSe usa un cable conectado directamente al dispositivo. Es ideal para configuración inicial.
TelnetRemotoPermite entrar por red, pero no es seguro porque no cifra la información.
SSHRemoto seguroPermite entrar por red de forma cifrada. Es la opción recomendada.
Puerto auxiliarRemoto por módemSe usaba para acceso fuera de banda; hoy es menos común.

Cable de consola

Se usa para configurar el router o switch cuando todavía no tiene dirección IP ni acceso remoto.

Switch con puertos Ethernet

Permite conectar varios equipos dentro de una misma red local.

Router Cisco

Equipo que permite interconectar redes y aplicar configuración mediante IOS.

Importante

Telnet es cómodo, pero inseguro. Si alguien captura el tráfico, puede leer usuario y contraseña. Por eso, en escenarios reales se recomienda usar SSH.

A-03

Modos de operación del IOS

El IOS tiene diferentes modos. Cada modo permite hacer cosas distintas. El símbolo del prompt te dice en qué nivel estás.

ModoPromptUso
EXEC de usuarioRouter>Permite revisar información básica.
EXEC privilegiadoRouter#Permite ejecutar comandos de verificación, guardar, reiniciar y entrar a configuración.
Configuración globalRouter(config)#Permite cambiar la configuración general del equipo.
Configuración de líneaRouter(config-line)#Permite configurar consola, VTY o auxiliar.
Configuración de interfazRouter(config-if)#Permite configurar una interfaz específica.
IOS – Cambio de modos
Router> enable                    ! Entra al modo privilegiado
Router# configure terminal        ! Entra a configuración global
Router(config)# interface g0/0 ! Entra a configurar una interfaz
Router(config-if)# exit       ! Regresa un nivel
Router(config)# end          ! Regresa al modo privilegiado
Comandos explicados paso a paso
enable
Te permite subir del modo usuario al modo privilegiado. Es como pedir permiso para usar comandos más importantes.
configure terminal
Abre el modo donde ya puedes cambiar la configuración del dispositivo.
interface g0/0
Selecciona una interfaz específica para configurarla.
exit
Regresa al modo anterior.
end
Te regresa directamente al modo privilegiado.
A-04

Ayuda de la CLI, abreviaciones y teclas rápidas

La CLI de Cisco ayuda mucho cuando todavía no recuerdas los comandos. Puedes usar el signo ?, escribir comandos incompletos y utilizar abreviaciones.

RecursoEjemplo¿Qué hace?
Ayuda sensible al contextoshow ?Muestra opciones disponibles después de show.
Verificación de sintaxisshow ip ?Muestra qué puede continuar después de show ip.
Abreviaciónconf tEquivale a configure terminal.
AutocompletarTabCompleta el comando si no hay ambigüedad.
HistorialFlecha arribaRecupera comandos escritos anteriormente.
A-05

Comandos de análisis de IOS

Los comandos de análisis no cambian la configuración: sirven para revisar el estado del dispositivo. Son como “herramientas de diagnóstico”.

ComandoUso
show running-configMuestra la configuración activa.
show startup-configMuestra la configuración guardada.
show ip interface briefMuestra resumen de interfaces, IP y estado.
show versionMuestra versión de IOS, memoria y datos del equipo.
pingPrueba conectividad hacia otro dispositivo.
